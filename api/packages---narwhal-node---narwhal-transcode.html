narwhal/transcode module

Provides stream decorators for translating bytes and characters.

  • DecoderReader
  • EncoderWriter

DecoderReader

A synchronous stream decorator that transforms byte reading into string reading along a given character set.

raw ByteReader
charset String
length Number

(optional) internal buffer size, defaults to 1024.

Properties:

read

Reads a string, either all remaning characters, or a chunk.
readMax

Reads a string, from the given maximum number of bytes.
readAll

Reads and transcodes all reamaning bytes to characters.
readLine

Reads a single line.
readLines

Reads all reamaining lines.
next

Reads the next line or throws StopIteration
iterator

Provides an object conforming to the iterator protocol for iterating the lines of the stream.
forEach

Calls a function with each remaining line of data from the stream.
copy

Copies all data from this stream to a given output stream.
close

Closes the underlying raw byte stream.

EncoderWriter

A synchronous stream decorator that transforms string writing into byte writing along a given character set.

raw Buffer
charset String
length Number

(optional) internal buffer size, defaults to 1024.

Properties:

write

Writes text to the stream.
flush

Flushes all written data through the underlying raw stream.
close

Closes the underlying raw stream.
