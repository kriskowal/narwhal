test module

Facilities for running and logging unit test modules.

An implementation and extension of the CommonJS Unit Testing 1.0 Specification's test module. http://wiki.commonjs.org/wiki/Unit_Testing/1.0

run

Runs and reports the results of a unit test or a tree of unit tests.

test Test ~ {test*: Function || This, setup?: Function, teardown?: Function}
  • a test object including test* functions or nested test * objects.
log Log ~ {pass, fail, error, section}

a log object for reporting test results.
returns Number

0 if all tests pass.
