request

Issues an HTTP request. The request may be any object that has method, path, headers and body properties.

  • method String is optional, defaults to "GET".
  • path String is optional, defaults to "/".
  • headers Object is optional, defaults to {}.
  • body is optional, defaults to []. Body must be an object with a forEach method that accepts a write callback. forEach may return a promise, and may send promises to write. body may be a promise.

The Q HTTP Server responder receives a Request object that is suitable for request, and request returns a Response suitable for returning to the Server.

returns Promise * Response
