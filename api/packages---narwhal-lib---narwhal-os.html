system

executes a given command, attached to this process's IO streams, and returns the exit status.

command Array or String

uses "/bin/sh -c" if the command is a string.
returns undefined

Number exit status

exec

Supported in Rhino and Node. On Rhino, acceptance of the command object and the search argument are not yet supported.

command String || Array || {args, env, search}

or "system" like object
env Object
search Boolean

defaults to true
throws Error

if execution resumes with the old program image.

exec0

Strictly built on Posix execv, execvp, and execvP, where the command is provided as the first argument and args[0] is the original path used to execute the command.

command String
args Array * String

optional
env Object * String

optional
search Boolean

optionally whether to use the env.PATH to search for command

status

executes a given command quietly and returns the exit status.

command Array or String

uses "/bin/sh -c" if the command is a string.
returns undefined

Number exit status

command

executes a given command and returns the standard output. If the exit status is non-zero, throws an Error.

command Array or String

uses "/bin/sh -c" if the command is a string.
returns undefined

String the standard output of the command

enquote

enquotes a string such that it is guaranteed to be a single argument with no interpolated values for a shell.

WARNING: as yet, this implementation only handles enquoting for Unix shell script style arguments. Further development is necessary to enquote and escape arguments on Windows.

parse

parses command line arguments

WARNING: this does not handle all of the edge cases of command line argument parsing, nor is suitable for general purpose argument enquoting on all platforms. It also will never be able to handle environment variable interpolation or other forms of shell quote expansion. This utility is used by Narwhal to pare arguments from system.env.NARWHAL_OPT.

command

{String} a command composed of space delimited, quoted, or backslash escaped arguments.
returns undefined

an Array of unquoted arguments.
