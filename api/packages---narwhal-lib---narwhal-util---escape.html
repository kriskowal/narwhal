escape

escapes all characters of a string that are special to JavaScript and many other languages. Recognizes all of the relevant control characters and formats all other non-printable characters as Hex byte escape sequences or Unicode escape sequences depending on their size.

Pass true as an optional second argument and escape produces valid contents for escaped JSON strings, wherein non-printable-characters are all escaped with the Unicode \u notation.
