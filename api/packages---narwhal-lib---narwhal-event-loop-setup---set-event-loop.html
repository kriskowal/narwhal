setEventLoop

Configures the system to use a particular event loop implementation. The event loop must support the event loop API:

  • setTimeout
  • clearTimeout
  • setInterval
  • clearInterval
  • enqueue is optional; if it is not provided, the event-loop module will provide one in terms of setTimeout(task, 0).
