operator

a decorator for functions that curry "polymorphically", that is, that return a function that can be tested against various objects if they're only "partially completed", or fewer arguments than needed are used.

this enables the idioms:

[1, 2, 3].every(lt(4)) eq true
[1, 2, 3].map(add(1)) eq [2, 3, 4]
[{}, {}, {}].forEach(set('a', 10))
