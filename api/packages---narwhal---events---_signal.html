constructor

addListener

arranges for a given observer function to be called whenever this signal is emitted. The calling of this observer function can in turn be observed by the returned signal.

observer

{Function} a function to call when this signal is emitted. All observers get called before the signal's "default action", if it has one. Observers are called on a first-come, first-served basis.
Event

{Event type} an optional Event type to be used for the returned Signal that defaults to Event.
returns undefined

a Signal that is emitted before the given observer.

removeListener

emit

emits an event with to all observers and the signal's default action, by applying each function with the event instance as the activation object and the forwarding the given arguments. Uses the event object to determine whether to continue propagating from observer to the next, and whether to apply the default action.

observer

var x = signal.observer(function (y, z) { print("signal was observed with " + y + ", " + z); }); x("a", 10); x("b", 20); x.emit(); signal was observed with a, 10 signal was observed with b, 20

returns undefined

a function that, when called with arbitrary arguments, arranges for the signal to be observed with a given function that receives those arguments.

emitter

returns Function

a function that emits this signal when it is called.

getObservers

returns Array

all observer functions in order. Modifications to the returned array will not affect the behavior of the signal.
