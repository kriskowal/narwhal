stringify

Converts an arbitrary value to a Query String representation.

Objects with cyclic references trigger an exception.

obj

{Variant} any arbitrary value to convert to query string
sep

{String} (optional) Character that should join param k=v pairs together. Default: "&"
eq

{String} (optional) Character that should join keys to their values. Default: "="
name

{String} (optional) Name of the current key, for handling children recursively.
