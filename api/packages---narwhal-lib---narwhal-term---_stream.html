Stream

Constructs a terminal output stream that translates literate color encodings like "\0blue(I'm blue\0)" into VT-100 escape sequences, maintaining independent stacks for foreground, background, and font weight so terminal settings are restored when they are out of scope.

Escape sequences are channeled to the stderr channel to preserve the consistency of the data written to stdout, such that it can be safely pipelined. The translating functions coordinate flushing of stdout and stderr to ensure that formatting is applied correctly.
