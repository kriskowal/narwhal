uuid

Generate a random uuid.

EXAMPLES:

// No arguments  - returns RFC4122, version 4 ID
js> Math.uuid()
"92329D39-6F5C-4520-ABFC-AAB64544E172"

// One argument - returns ID of the specified length
js> Math.uuid(15)     // 15 character ID (default base=62)
"VcydxgltxrVZSTV"

// Two arguments - returns ID of the specified length, and radix. (Radix must be <= 62)
js> Math.uuid(8, 2)  // 8 character ID (base=2)
"01001010"
js> Math.uuid(8, 10) // 8 character ID (base=10)
"47473046"
js> Math.uuid(8, 16) // 8 character ID (base=16)
"098F4D35"
length Number

(optional) the desired number of characters.
radix Number

(optional) the number of allowable values for each character.
