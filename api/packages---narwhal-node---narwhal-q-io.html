narwhal/q-io module

A Node implementation for promise-based IO.

  • WrappedTextReader
  • WrappedTextWriter

WrappedTextReader

Wraps a Node readable stream, providing an API similar to a Narwhal synchronous io stream except returning Q promises for long latency operations.

stream

any Node readable stream
returns Promise * WrappedTextReader

a promise for the text stream reader.

Properties:

read

Reads all of the remaining data from the stream.
forEach

Reads and writes all of the remaining data from the stream in chunks.

WrappedTextWriter

Wraps a Node writable stream, providing an API similar to Narwhal's synchronous io streams, except returning and accepting promises for long-latency operations.

stream

any Node writable stream
returns Promise * WrappedTextWriter

a promise for the text writer.

Properties:

write

Writes content to the stream.
flush

Waits for all data to flush on the stream.
close

Closes the stream, waiting for the internal buffer to flush.
destroy

Terminates writing on a stream, closing before the internal buffer drains.
