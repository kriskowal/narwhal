Sandbox

Creates a require function, and arranges for modules to be executed and their exports memoized, in a lexical scope that includes:

  • require(id) with support for identifiers relative to the calling module.
  • require.loader for direct access to the module loader, which can be used in nested sandboxes.
  • require.force(id)
  • require.once(id, scope) to execute but not memoize a module, with an optional object that owns additional free variables to inject into the module's lexical scope.
