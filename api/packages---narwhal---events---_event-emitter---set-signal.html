setSignal

adds a Signal slot to this event emitter with a given name.

name

{String} the name of the signal slot. Only one signal can be created for each name.
signal

an optional function or Signal object
Event

an optional event type that will be instantiated and passed as the activation object to each observer each time the signal emits an event.
returns undefined

a signal, albeit one wrapped around the given function.
