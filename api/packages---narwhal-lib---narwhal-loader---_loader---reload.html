reload

Forces a module to be reloaded, setting or replacing the module factory function in the module factory memo. This function is called by load internally to populate the memo and may be called externally to freshen the memo.

path String

an optional path to provide a hint about where to find the text corresponding to the given topId, provided merely to avoid recomputation.
